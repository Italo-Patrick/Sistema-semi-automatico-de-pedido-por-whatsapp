Catálogo Digital com Pedido via WhatsApp

Este projeto é um catálogo digital simples desenvolvido com HTML, CSS e JavaScript, que permite ao cliente selecionar produtos e enviar o pedido diretamente para o WhatsApp do comerciante.

O objetivo é facilitar o processo de pedidos para pequenos negócios, tornando o atendimento mais rápido e organizado.

 Funcionalidades

* Listagem de produtos a partir de um arquivo JSON
* Separação por categorias (ex: sanduíches, bebidas)
* Seleção de quantidade por produto
* Cálculo automático do valor total
* Formulário para nome, endereço e observações
* Geração automática da mensagem de pedido
* Envio direto para o WhatsApp

 Como funciona

1. Os produtos são carregados via `produtos.json`
2. O usuário escolhe os itens e quantidades
3. O sistema calcula o total em tempo real
4. Ao enviar, uma mensagem formatada é gerada
5. O pedido é aberto diretamente no WhatsApp

 Estrutura do projeto

/imgs              → imagens dos produtos
produtos.json      → dados dos produtos
index.html         → estrutura da página
style.css          → estilos
script.js          → lógica da aplicação

 Personalização

Para adaptar o sistema para outro negócio:

* Alterar os produtos em `produtos.json`
* Atualizar o número do WhatsApp no arquivo `script.js`
* Substituir imagens na pasta `/imgs`
* Ajustar textos conforme necessário

 Responsividade

O projeto foi desenvolvido com foco em dispositivos móveis (mobile-first), garantindo boa experiência em smartphones.

 Observações

Este projeto não utiliza backend ou banco de dados.
Todos os dados são processados no lado do cliente.

 Tecnologias utilizadas

* HTML
* CSS
* JavaScript
